Welcome to Print2Paper4VSCode Discussions. This is the place to ask questions, share how you use the extension, and suggest ideas.

**Bug reports belong in [Issues](<https://github.com/appliedmedia/print2paper4vscode/issues>)** — if something is broken or not working as documented, open an issue there so it gets tracked and fixed.

**Discussions are for everything else:**

* Questions about setup, keybindings, or page size configuration
* Feature requests and ideas for future improvements
* Sharing workflows or use cases
* General feedback

We are a small team and aim to respond within a few days. Thanks for using Print2Paper4VSCode.
