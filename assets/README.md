# assets

Static assets shipped with the extension or used in the marketplace listing.

## p2p4vsc-screenshots

Toolbar screenshots used in the marketplace walkthrough (`contributes.walkthroughs` in `config/template.package.json`) and in `docs/MARKETPLACE.md`. One PNG per toolbar menu: Print, Page, Theme, Text, Zoom, Markdown mode, About.

These images are included in the published VSIX via the `.vscodeignore` exception `!assets/p2p4vsc-screenshots/**`.

## Other files

Badge SVGs and CI status images referenced from the root `README.md`.
